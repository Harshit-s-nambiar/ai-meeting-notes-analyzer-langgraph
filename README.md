# AI Meeting Notes Analyzer using LangGraph

AI-powered meeting notes analyzer using LangGraph, LLMs, and workflow automation.

---

## Project Overview

This project analyzes meeting transcripts using LLM-powered workflows built with LangGraph.

The system processes meeting conversations and automatically generates:
- meeting summaries
- key discussion points
- action items
- structured outputs

The project demonstrates workflow orchestration, prompt engineering, and AI automation using modern LLM frameworks.

---

## Features

- AI-powered meeting summarization
- Transcript analysis
- Action item extraction
- Workflow automation using LangGraph
- Structured output generation
- Notebook and script-based execution

---

- Python

## Installation

```bash
git clone https://github.com/Harshit-s-nambiar/ai-meeting-notes-analyzer-langgraph.git

cd ai-meeting-notes-analyzer-langgraph

pip install -r requirements.txt
```

---

## Run the Project

Run the Python application:

```bash
python app.py
```

Or run the notebook:

```bash
notebooks/meeting_notes_analyzer.ipynb
```
